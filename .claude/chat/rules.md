# Chat Rules — System Prompt & Behavior

## فایل مرتبط
`backend/routers/chat.py` — متغیر `SYSTEM_PROMPT`

## قوانین سخت‌گیری (off-topic)
- Claude باید هر سوال غیرمرتبط با پروژه وب رو **فوری و بدون توضیح اضافه** رد کنه
- جواب رد کردن همیشه کوتاه باشه: فقط یه جمله + برگشت به پروژه
- اگه کاربر اصرار کرد، Claude نباید کوتاه بیاد — هر بار همان جواب کوتاه
- موضوعات ممنوع: آشپزی، سیاست، سرگرمی، اخبار، هر چیزی که به وب/پروژه ربط نداشته باشه
