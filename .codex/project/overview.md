# Project Overview

## چشم‌انداز

**FreelioAI** یک پلتفرم ارائه Agent هوشمند برای فریلنسرهاست.

هر فریلنسر یک چت‌بات اختصاصی دریافت می‌کند که می‌تواند آن را
به‌صورت Embed در وب‌سایت یا لندینگ‌پیج خود قرار دهد.
مشتریان از طریق این چت‌بات نیازمندی‌های پروژه‌شان را توضیح می‌دهند،
Agent با داده‌های فریلنسر مکالمه را مدیریت می‌کند و در نهایت
یک درخواست ساختاریافته برای فریلنسر ثبت می‌شود.

> **وضعیت فعلی:** MVP در حال توسعه — فلوی intake تا پرداخت اولیه پیاده شده.
> ماژول‌های Project Space و AI Agent پروژه هنوز ساخته نشده‌اند.

---

## فلوی کامل محصول

### ۱ — Intake (چت‌بات روی سایت فریلنسر)

```
مشتری وارد سایت فریلنسر می‌شود
    ↓
ویجت چت‌بات load می‌شود (embed از FreelioAI)
    ↓
Agent با داده‌های فریلنسر مکالمه را شروع می‌کند:
  - نوع خدمات فریلنسر
  - محدوده قیمت
  - فرآیند کاری
  - سوالات موردنیاز
  - شرایط پروژه
    ↓
نیازمندی‌ها مرحله‌به‌مرحله جمع‌آوری می‌شود
    ↓
درخواست ساختاریافته + محدوده تقریبی قیمت تولید می‌شود
    ↓
شماره تماس مشتری دریافت و OTP تایید می‌شود
    ↓
درخواست برای فریلنسر ثبت و ارسال می‌شود
```

### ۲ — Proposal (پنل فریلنسر)

```
فریلنسر درخواست را در پنل مشاهده می‌کند
    ↓
AI خلاصه دقیق نیازمندی‌ها + پیشنهاد قیمت اولیه را نمایش می‌دهد
    ↓
فریلنسر تعریف می‌کند:
  - مبلغ نهایی
  - Milestoneها
  - Proposal رسمی
    ↓
Proposal برای مشتری ارسال می‌شود
```

### ۳ — تایید و پرداخت (پنل مشتری)

```
مشتری Proposal + Milestoneها را بررسی می‌کند
    ↓
در صورت تایید، پروژه قبول می‌شود
    ↓
Project Space اختصاصی ایجاد می‌شود
    ↓
مشتری پرداخت مرحله‌ای (per Milestone) انجام می‌دهد (Zarinpal)
```

### ۴ — Project Space (فضای اختصاصی پروژه)

```
فضای مشترک مشتری و فریلنسر شامل:
  ├── چت یک‌به‌یک (مستقیم بین طرفین)
  ├── مدیریت Milestoneها (وضعیت، تایید، پرداخت)
  ├── مدیریت پرداخت‌ها
  ├── ارسال و دریافت فایل و مستندات
  └── پیگیری وضعیت کلی پروژه
```

### ۵ — AI Agent پروژه

```
وقتی فریلنسر آنلاین نیست:
    ↓
Agent با استفاده از:
  - داده‌های ثبت‌شده پروژه
  - تاریخچه گفتگوها
  - Milestoneهای تعریف‌شده
    ↓
به سوالات مشتری پاسخ می‌دهد
تغییرات درخواستی را شناسایی و به فریلنسر گزارش می‌دهد
در صورت نیاز Task جدید برای فریلنسر تعریف می‌کند
```
