# پلن محتوا — FreelioAI

> **هدف:** جذب فریلنسرهای ایرانی از طریق محتوای ارزش‌آفرین، نه تبلیغ مستقیم

---

## ۱. زاویه محتوایی — چه چیزی فریلنسر ایرانی را engage می‌کند

### چه چیزی کار می‌کند

**الف — درد واقعی را لمس کن**
فریلنسر ایرانی وقتی یه چیزی می‌بیند که "دقیقاً همین مشکل منه!" احساس کند، share می‌کند.
مثال: "وقتی مشتری ساعت ۱۱ شب می‌گه: یه چیز کوچیک داشتم..."

**ب — داستان واقعی، نه آمار خشک**
نه: "۷۰٪ فریلنسرها مشکل scope دارند"
بله: "سه هفته روی یه پروژه کار کردم. هفته آخر مشتری گفت: فکر می‌کردم این هم توش بود..."

**ج — آموزش عملی، نه نظری**
- "بریف‌گیری را این‌طور بپرس" (نه "بریف‌گیری مهم است")
- template، checklist، نمونه واقعی

**د — تایید اجتماعی بومی**
- testimonial از فریلنسر ایرانی، به زبان فارسی، با مشکل واقعی ایرانی
- نه ترجمه تجربه غربی

**ه — طنز آشنا**
فریلنسر ایرانی با طنز مرتبط بیشتر engage می‌کند.
مثال: "مشتری: 'خب این چیزه که... می‌دونی دیگه' / فریلنسر: نه. نمی‌دونم."

---

### چه چیزی کار نمی‌کند

- کلیشه‌های انگلیسی ترجمه‌شده ("ارزش پیشنهادی منحصربه‌فرد شما")
- ادعاهای بزرگ بدون اثبات
- محتوای عمومی که برای همه است و برای هیچ‌کس نیست
- post های فقط برای SEO

### پست ۱ — "قاتل خاموش درآمد فریلنسر"
**فرمت:** کاروسل اینستاگرام / Thread توییتر
**زاویه:** درد scope creep
**محتوا:**

```
اسلاید ۱: "هر هفته چند ساعت از درآمدت رو دست می‌دی بدون اینکه بدونی؟"

اسلاید ۲: اسمش scope creep است
  مشتری یه چیز می‌خواد
  وسط راه یه چیز اضافه می‌شه
  آخر کار ۲۰ چیز شده
  تو همه رو انجام دادی — پول فقط اولی رو گرفتی

اسلاید ۳: چرا قبول می‌کنیم؟
  "آدم بدی نباشم"
  "یه بار دیگه، آخرین بار"
  "نمی‌خوام مشتری ناراضی بشه"

اسلاید ۴: راه‌حل: بریف مکتوب قبل از شروع
  اگه در ابتدا بنویسی "این هست، این نیست"
  هر تغییر بعدی = پروپوزال جدید

اسلاید ۵: یه سوال داری؟ توی کامنت بنویس
```

**هشتگ‌ها:** #فریلنسر #برنامه‌نویس_فریلنسر #طراح_گرافیک #scope_creep

### پست ۲ — "چقدر واقعاً در ساعت درآمد داری؟"
**فرمت:** کاروسل / Thread
**زاویه:** محاسبه درآمد واقعی vs. اسمی

```
اسلاید ۱: "فریلنسر گفت: ماهی ۲۰ میلیون در می‌آرم"
  بریم حساب کنیم...

اسلاید ۲:
  ۲۰ میلیون ÷ ۲۲ روز کاری = ۹۰۹ هزار تومان/روز
  ÷ ۸ ساعت = ۱۱۳ هزار تومان/ساعت
  خوبه؟ صبر کن...

اسلاید ۳: ساعت‌های پنهان:
  بریف‌گیری: ۱ ساعت/پروژه × ۵ پروژه = ۵ ساعت
  رفع ابهام: ۲ ساعت/هفته
  پیام‌های جواب‌دادن: ۳ ساعت/هفته
  جمعاً: ۱۰ ساعت/هفته که پول نمی‌گیری

اسلاید ۴: حساب واقعی:
  ۲۰ میلیون ÷ (۴۰ساعت کاری - ۱۰ساعت غیرمولد) = ۱۵۰ هزار/ساعت واقعی
  نه ۱۱۳ هزار

اسلاید ۵: این ۱۰ ساعت رو پس بگیر
```

---

### پست ۳ — "قرارداد واتساپی یعنی قرارداد نداری"
**فرمت:** Thread توییتر
**زاویه:** ریسک قرارداد غیررسمی

```
Thread:
۱/ "قرارداد" تو واتساپ = قرارداد نداری.

۲/ چرا؟
- تاریخچه پاک می‌شه
- اسکرین‌شات قابل رد است
- "من این رو نگفتم" — و تو نمی‌تونی ثابت کنی

۳/ واقعی: یه طراح ۶ ماه پیش یه پروژه ۳۰ میلیونی داشت
  آخر کار مشتری ادعا کرد ۲ صفحه اضافه هم "جزو پروژه" بود
  دعوا، استرس، ۲ میلیون تخفیف داد که تموم شه

۴/ راه‌حل ساده:
  قبل از شروع، یه سند یا پروپوزال رسمی
  "اگه امضا نشد، شروع نمی‌کنم"
  این جمله بیشتر از هر ابزاری نجاتت می‌ده

۵/ سوال: تا حالا بدون قرارداد رسمی ضرر کردی؟
```

### پست ۴ — "۷ سوالی که باید قبل از هر پروژه بپرسی"
**فرمت:** کاروسل / Thread آموزشی
**زاویه:** عملی، قابل‌استفاده فوری

```
۱. هدف اصلی این پروژه چیست؟
   (نه "یه سایت بساز" — چرا این سایت؟ چه نتیجه‌ای انتظار داری؟)

۲. مخاطب اصلی کیست؟
   (اگه نمی‌داند = مشکل اول پیدا شد)

۳. چه چیزی باید حتماً باشد؟ چه چیزی نباید باشد؟

۴. مهلت واقعی چیست؟ چرا آن تاریخ؟

۵. تصمیم‌گیرنده نهایی کیست؟
   (بزرگ‌ترین اشتباه: با اجرایی صحبت کنی، با مدیر تغییر کند)

۶. بودجه تقریبی در ذهن داری؟
   (اگه نگفت = احتمالاً با قیمتت مشکل خواهد داشت)

۷. اگه این پروژه ۱۰۰٪ موفق باشد، چه تغییری در کسب‌وکارت می‌بینی؟
   (این سوال scope را شفاف می‌کند)

این ۷ سوال رو ذخیره کن — قبل از هر پروژه بخوان.
```

### پست ۵ — "چرا مشتری‌ات فکر می‌کنه گرونی؟"

**زاویه:** قیمت‌گذاری بر اساس ارزش، نه ساعت

```
مشتری نمی‌گه "گرونه" چون گرونه.

وقتی می‌گی:
"لندینگ پیج که مشتری‌هاتو از تبلیغ به خرید هدایت می‌کنه

  برای کسب‌وکار ۱۰ میلیونی، این یعنی ۳ میلیون اضافه در ماه اول
  قیمت ما: ۴ میلیون"

مشتری می‌شنوه: "سرمایه‌گذاری که در یه ماه برمی‌گرده"

تفاوت در قیمت نیست. در framing است.
```

```

نتیجه:
آخرین پروژه‌اش دقیقاً طبق scope اولیه تمام شد.
نه اضافه‌کاری، نه چانه‌زنی.

Quote: "[یک جمله از طراح]"
```

## ۶. نکات اجرایی

**قانون ۸۰/۲۰:**
- ۸۰٪ محتوا: ارزش‌آفرین (آموزشی، داستان، درد)
- ۲۰٪ محتوا: معرفی محصول (نرم، نه مستقیم)

**بهترین زمان پست در ایران:**
- اینستاگرام: ۱۹-۲۱ (شب، بعد از شام)
- توییتر: ۱۰-۱۲ صبح و ۲۰-۲۲ شب
